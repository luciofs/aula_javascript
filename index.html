FUNÇÕES CLÁSSICAS:

function sum(a, b){

return a + b;

}

console.log(sum(5, 10));

FUNÇÕES ANÔNIMAS:

(a função é anônima quando ela é atibuída a uma variável ou quando passada como parâmetro para uma outra função)

var sum = function(a, b){

return a + b;

}

console.log(sum(22, 15));

ARROW FUNCTIONS

var sum = (a, b) => a + b;

console.log(sum(12, 45));
